# Angular 19 → 21 Upgrade Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bump `frontend/` from Angular 19 to 21, adopt the stable code-quality and rendering wins along the way, and prove the rendering claim with an empirical before/after number.

**Architecture:** Linear-by-phase. Eight commits on a single branch (`chore/angular-21-upgrade`), each commit semantically scoped, each commit reverts cleanly. Two highest-risk commits (the v21 bump and the zoneless cutover) are isolated as standalone units. A throwaway perf-trace script captures one number before P1 starts and one after P6 lands; the delta goes in the P7 commit message and the source design doc.

**Tech Stack:** Angular 19 → 21, TypeScript 5.6 → 5.9, `@angular/build` application builder, Karma + Jasmine for unit tests, Playwright for E2E and for the perf trace, devcontainer-only test gate (Linux node_modules), per-package coverage ratchet at 3.0% tolerance.

**Source design:** `docs/superpowers/specs/2026-05-04-angular-21-upgrade-design.md` (master, sha `3e8b2bb`).

- [ ] **Step 4: Add `loadFixture()` to the workspace store**

Add a method that bypasses Firestore and sets the internal state signals directly from a passed-in fixture. The shape below is a template — adapt the signal names to match what you observed in Step 3.

```ts
// inside packages/editor/src/services/workspace-store.ts (or equivalent)

export interface FixtureProject {
  readonly projectId: string;
  readonly pages: ReadonlyArray<{
    readonly id: string;
    readonly elements: ReadonlyArray<unknown>; // use the existing element type
  }>;
}

// Inside the WorkspaceStore class:
public loadFixture(fixture: FixtureProject): void {
  // Cancel any in-flight Firestore subscriptions if the store has them.
  // Set the state signals directly so the editor renders the fixture
  // without any network round-trip.
  this._projectId.set(fixture.projectId);
  this._pages.set(fixture.pages);
  this._activePageId.set(fixture.pages[0]?.id ?? null);
  // ... mirror whatever shape the store currently expects.
}
```

If the existing store has no setters (signals defined as private and only mutated via internal effects), make `loadFixture` the one entry point that breaks that encapsulation, and add a one-line comment: `// Test/perf-fixture-only entry point; bypasses Firestore.`

```ts
const isPerfMode = (): boolean =>
  typeof window !== 'undefined' && window.location.search.includes('perf=1');

const baseRoutes: Routes = [
  // ... existing routes unchanged
];

const perfRoutes: Routes = [
  {
    path: 'perf-fixture',
    loadComponent: () => import('./perf-fixture/perf-fixture.component').then(m => m.PerfFixtureComponent),
  },
];

const routes: Routes = isPerfMode() ? [...baseRoutes, ...perfRoutes] : baseRoutes;
```

The `loadComponent` lazy import ensures the perf component bundle isn't fetched unless the user is at `?perf=1`. Production cost: a few bytes in the routes array.

- [ ] **Step 12: Create the perf trace script**

Create `frontend/scripts/editor-perf-trace.mjs`:

```js
import { chromium } from '@playwright/test';
import { spawn } from 'node:child_process';
import { writeFileSync, mkdirSync } from 'node:fs';
import { resolve, dirname } from 'node:path';
import { fileURLToPath } from 'node:url';

const __dirname = dirname(fileURLToPath(import.meta.url));
const SAMPLES = 5;
const BASELINE_DIR = resolve(__dirname, 'perf-baselines');

const outputName = process.argv[2];
if (!outputName) {
  console.error('Usage: node scripts/editor-perf-trace.mjs <output-name>');
  console.error('  e.g. node scripts/editor-perf-trace.mjs 19.x');
  process.exit(2);
}

mkdirSync(BASELINE_DIR, { recursive: true });

console.log('[perf] starting dev server...');
const dev = spawn('npm', ['start'], {
  cwd: resolve(__dirname, '..'),
  stdio: ['ignore', 'pipe', 'pipe'],
});
const ready = new Promise((res, rej) => {
  let buf = '';
  const onData = (chunk) => {
    buf += chunk.toString();
    if (buf.includes('Local:') || buf.includes('localhost:4200')) res();
  };
  dev.stdout.on('data', onData);
  dev.stderr.on('data', onData);
  setTimeout(() => rej(new Error('dev server did not start in 60s')), 60_000);
});
await ready;
await new Promise((r) => setTimeout(r, 2000)); // settle

const samples = [];

for (let i = 0; i < SAMPLES; i++) {
  console.log(`[perf] sample ${i + 1}/${SAMPLES}`);
  const browser = await chromium.launch();
  const context = await browser.newContext();
  const page = await context.newPage();
  const cdp = await context.newCDPSession(page);

  await cdp.send('Performance.enable');

  await page.addInitScript(() => {
    window.__perf = { longTasks: 0, frames: 0 };
    new PerformanceObserver((list) => {
      window.__perf.longTasks += list.getEntries().length;
    }).observe({ type: 'longtask', buffered: true });
    const tickFrame = () => {
      window.__perf.frames++;
      requestAnimationFrame(tickFrame);
    };
    requestAnimationFrame(tickFrame);
  });

  await page.goto('http://localhost:4200/perf-fixture?perf=1');
  await page.waitForLoadState('networkidle');
  await page.waitForSelector('mk-editor', { timeout: 10_000 });

  const startMetrics = await cdp.send('Performance.getMetrics');

  // Drag text-1 200px right
  const target = page.locator('mk-editor [data-perf-id="text-1"]').first();
  const box = await target.boundingBox();
  if (box) {
    await page.mouse.move(box.x + box.width / 2, box.y + box.height / 2);
    await page.mouse.down();
    await page.mouse.move(box.x + 200, box.y + box.height / 2, { steps: 20 });
    await page.mouse.up();
  }

  // Type into text-1's contenteditable
  const textArea = page.locator('mk-editor [data-perf-id="text-1"] [contenteditable]').first();
  if (await textArea.count()) {
    await textArea.click();
    await page.keyboard.type('hello world', { delay: 30 });
  }

  const endMetrics = await cdp.send('Performance.getMetrics');

  const delta = (name) => {
    const start = startMetrics.metrics.find((m) => m.name === name)?.value ?? 0;
    const end = endMetrics.metrics.find((m) => m.name === name)?.value ?? 0;
    return +(end - start).toFixed(6);
  };

  const perfWindow = await page.evaluate(() => window.__perf);

  samples.push({
    sampleIndex: i,
    scriptDurationMs: +(delta('ScriptDuration') * 1000).toFixed(3),
    taskDurationMs: +(delta('TaskDuration') * 1000).toFixed(3),
    layoutCount: delta('LayoutCount'),
    recalcStyleCount: delta('RecalcStyleCount'),
    layoutDurationMs: +(delta('LayoutDuration') * 1000).toFixed(3),
    recalcStyleDurationMs: +(delta('RecalcStyleDuration') * 1000).toFixed(3),
    longTasks: perfWindow.longTasks,
    frames: perfWindow.frames,
  });

  await browser.close();
}

const median = (arr) => {
  const sorted = [...arr].sort((a, b) => a - b);
  // drop high + low, average the middle three
  const middle = sorted.slice(1, -1);
  return middle.reduce((a, b) => a + b, 0) / middle.length;
};

const result = {
  capturedAt: new Date().toISOString(),
  outputName,
  interactionScriptVersion: 1,
  samples,
  median: {
    scriptDurationMs: +median(samples.map((s) => s.scriptDurationMs)).toFixed(3),
    taskDurationMs: +median(samples.map((s) => s.taskDurationMs)).toFixed(3),
    layoutCount: +median(samples.map((s) => s.layoutCount)).toFixed(3),
    longTasks: +median(samples.map((s) => s.longTasks)).toFixed(3),
    frames: +median(samples.map((s) => s.frames)).toFixed(3),
  },
};

const outPath = resolve(BASELINE_DIR, `${outputName}.json`);
writeFileSync(outPath, JSON.stringify(result, null, 2));
console.log(`[perf] wrote ${outPath}`);
console.log(`[perf] median scriptDurationMs: ${result.median.scriptDurationMs}`);

dev.kill();
process.exit(0);
```
